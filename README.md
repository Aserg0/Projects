# Portfolio

Я совершенствую свои навыки работы с Python и его библиотеками(Numpy, Pandas, Matplotlib, Plotly, SciPy, Seaborn, Phik, Folium) и SQL, создав портфолио проектов Яндекс.Практикум.

Навыки, которые я использую в работе с данными:

- необходимая предобработка данных (работа с текстом, явные/неявные дубликаты),
- исследовательский анализ (аномалии/выбросы) данных,
- выявление зависимостей и связей в данных для получения инсайтов, 
- визуализация данных (диаграммы, графики),
- расчет бизнес-показателей и ключевых метрик, 
- построение маркетинговых воронок,
- определение метрик, улучшением которых нужно заниматься,
- анализ данных A/B-тестирования,
- выбор метода проверки гипотез,
- приоритизация гипотез,
- статистический анализ к результатам A/B-теста для проверки гипотез,
- построение дашбордов,
- обработка, выгрузка данных с помощью SQL-запросов.
